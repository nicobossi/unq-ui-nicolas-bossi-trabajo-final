# Trabajo Final Integrador para la Materia Construcción de Interafaces de Usuario

# Preguntados

¿Cómo descargar el proyecto?

    1. Instalar  **Node.js**
    2. Copiar la URL del repositorio en la sección de código.
    3. Ejecutar el siguiente comando para descargar el proyecto: 
    **git clone <URL>**

¿Cómo empezar a jugar?

    1. Ejecutar el comando **npm install** para agregar todas las dependencias necesarias del proyecto.
    2. Ejecutar el comando **npm run dev** para correr el proyecto de forma local (en modo de desarrollo).

# Funcionalidades implementadas

    - Selector de dificultades.
    - Interfaz para cada pregunta con sus respectivas opciones.
    - Marcador de respuesta Correcta o Incorrecta.
    - Contador de preguntas respondidas.
    - Contador de respuestas correctas.
    - Interfaz de partida finaliza.
    - Interfaz de carga.
    - Interfaz para mostrar errores en la carga de datos.
    - Posibilidad de jugar varias partidas.
    - Interfaz Responsive.
