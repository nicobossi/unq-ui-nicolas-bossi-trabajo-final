# Preguntas y Respuestas

## Trabajo Final Integrador para la Materia Construcción de Interfaces de Usuario

### ¿Cómo descargar el proyecto?

1. Copiar la URL del repositorio en la sección **<> Code** de GitHub.  
2. Abrir alguna terminal y ejecutar el siguiente comando para descargar el proyecto:  
   <strong>`git clone <URL>`</strong>
3. Ejecutar el siguiente comando en la terminal para ir al directorio del repositorio clonado:
   <strong>`cd unq-ui-nicolas-bossi-trabajo-final`</strong>


### ¿Cómo empezar a jugar?

1. Instalar **Node.js**.
2. Ejecutar el comando <strong>`npm install`</strong> para agregar todas las dependencias necesarias del proyecto.
3. Ejecutar el comando <strong>`npm run dev`</strong> para correr el proyecto de forma local (en modo de desarrollo).

### Funcionalidades implementadas

- Selector de dificultades.
- Interfaz para cada pregunta con sus respectivas opciones.
- Marcador de respuesta Correcta e Incorrecta.
- Contador de preguntas respondidas.
- Contador de respuestas correctas.
- Interfaz de partida finalizada.
- Interfaz de carga.
- Interfaz para mostrar errores en la carga de datos.
- Posibilidad de jugar varias partidas.
- Interfaz Responsive.

### Tecnologías Utilizadas

- React 19
- TypeScript
- Axios
- Vite
